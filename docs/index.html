What's inside

Five modules of tiny, one-concept-per-file examples. Build it by hand, then meet the prebuilt.

  1. Single agent — bare state graph → hand-built ReAct loop → create_react_agent.
  2. Tool use & failure modes★ the centerpiece — 15 examples: errors, retries, timeouts, rate limits, infinite loops, hallucinated tools, human-in-the-loop.
  3. Multi-agent — shared state, handoffs, network routing.
  4. Orchestrator — supervisor, map-reduce fan-out, hierarchical teams.
  5. Advanced — memory, streaming, subgraphs, structured output, tracing.
