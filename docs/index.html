02The failure-modes matrix

The centerpiece. Real agentic tool use fails in many ways — most of them quietly. Module 2 reproduces each one deterministically and shows the LangGraph mechanism that handles it.

#FailureWhat triggers itMechanism / fix
01–03Tool raises an exceptiona tool throwsToolNode(handle_tool_errors=…) catch → ToolMessage, custom string, or fail-fast
04Invalid / missing argumentsmodel passes a non-intpydantic ValidationError surfaced as a ToolMessage — a self-correction signal
05Hanging / slow toola tool sleeps foreverasyncio.wait_for inside the tool — LangGraph won’t kill a hung node
06Transient errorflaky network.with_retry() backoff + jitter on the runnable
07Permanent failureprimary always down.with_fallbacks([…]) degrade to a backup
08Rate limit (429)provider throttles.with_retry + InMemoryRateLimiter
09Infinite tool-calling loopmodel never stopsrecursion_limit → GraphRecursionError (raw graphs)
10…handled gracefullysame loopcatch the error, or the prebuilt’s built-in remaining_steps graceful stop
11Hallucinated tool namecalls a tool that doesn’t existToolNode returns a “tool not found” ToolMessage — no crash
12Wrong tool / ignored resultanswers from “memory”detect via the transcript / stream_mode="updates"; mitigate with prompt + response_format
13Non-determinismsame input, different outputtemperature=0 + the deterministic fake model
14–15Destructive action, no guardraildelete prodinterrupt_before & interrupt() + checkpointer → human approval, then resume

03What the code looks like

A few representative slices. Every snippet below is real, runs offline, and is covered by a test.

m1 · the ReAct loop, by hand
def should_continue(state):
    last = state["messages"][-1]
    # the AIMessage asked for a tool? keep looping.
    return "tools" if last.tool_calls else END

g = StateGraph(MessagesState)
g.add_node("agent", agent_node)        # model.bind_tools([...])
g.add_node("tools", ToolNode([calculator]))
g.add_edge(START, "agent")
g.add_conditional_edges("agent", should_continue,
                        {"tools": "tools", END: END})
g.add_edge("tools", "agent")           # the cycle
graph = g.compile()
m2 · human-in-the-loop on a destructive tool
agent = create_react_agent(
    model, [dangerous_delete],
    checkpointer=MemorySaver(),
    interrupt_before=["tools"])        # pause before any tool

cfg = {"configurable": {"thread_id": "t1"}}
agent.invoke({"messages": [HumanMessage("delete prod-db")]}, cfg)
agent.get_state(cfg).next              # ('tools',)  ← paused
# ...human reviews, then approves:
agent.invoke(None, cfg)               # resumes & runs it

m4 · map-reduce fan-out with Send
def planner(state):
    # one dynamic branch per work item
    return [Send("worker", {"item": x}) for x in state["items"]]

def worker(state):
    return {"results": [process(state["item"])]}   # reducer merges

g.add_conditional_edges(START, planner, ["worker"])
g.add_edge("worker", "synthesize")
